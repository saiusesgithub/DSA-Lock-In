class Solution {
    public boolean isAnagram(String s, String t) {
        if (s.length() != t.length()) {
            return false;
        }
        
        char[] arr1 = s.toCharArray();
        char[] arr2 = t.toCharArray();
        Arrays.sort(arr1);
        Arrays.sort(arr2);

        for (int i =0;i<arr1.length;i++) {
            if (arr1[i] != arr2[i]) {
                return false;
            }
        }
        return true;

    }
}

## Problem 2

### 268. Missing Number
Easy

never did this problem before but felt easy enough after reading

first thought -> first sort the array and then loop from 0 to n-1 , if i != nums[i] then that number not found , if array ends (goes until nums.length-1) , then not found number is nums.length (cuz one missing number for sure and nums.length is not checked) 
```java
class Solution {
    public int missingNumber(int[] nums) {
        Arrays.sort(nums);
        for (int i =0;i<nums.length;i++) {
            if (i != nums[i]) {
                return i;
            }
        }
        return nums.length;
    }
}
```

```java
class Solution {
    public int missingNumber(int[] nums) {
        int n = nums.length;
        int expectedSum = ( n * (n+1) ) / 2;
        int actualSum = 0;
        for (int i =0;i<nums.length;i++) {
            actualSum = actualSum + nums[i];
        }
        return expectedSum - actualSum;
    }
}
```
