## Problem 1 - 

### 160. Intersection of Two Linked Lists
Easy

came up with this - 

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        HashSet<ListNode> set = new HashSet<>();
        ListNode pointerA = headA;
        while (pointerA != null) {
            set.add(pointerA);
            pointerA = pointerA.next;
        }
        ListNode pointerB = headB;
        while (pointerB != null) {
            if (!set.add(pointerB)) {
                return pointerB;
            }
            pointerB = pointerB.next;
        }
        return null;
    }
}
```

### 234. Palindrome Linked List
Easy

immediately came up with this , i know its not the most optimal version but good starting point
reversing the linked list and comparing each node using 2 pointer nodes
and in the end , if both the pointer nodes are not null , then it means that both the linked lists are not of same size

```java
class Solution {
    public boolean isPalindrome(ListNode head) {
        ListNode reverse = new ListNode(head.val);
        ListNode temp = reverse;
        ListNode pointer = head.next;
        while (pointer!=null) {
            temp = reverse;
            reverse = new ListNode(pointer.val,temp);
            pointer = pointer.next;
        }

        ListNode pointer1 = head;
        ListNode pointer2 = reverse;
        while (pointer1 != null && pointer2 != null) {
            if (pointer1.val != pointer2.val) {
                return false;
            }
            pointer1 = pointer1.next;
            pointer2 = pointer2.next;
        }
        if (pointer1 != null || pointer2!=null) {
            return false;
        }
        return true;

    }
}
```

```java
class Solution {
    public boolean isPalindrome(ListNode head) {
        Stack<Integer> stack = new Stack<>();
        ListNode slow = head;
        ListNode fast = head;
        while (fast != null && fast.next!=null) {
            stack.push(slow.val);
            slow = slow.next;
            fast = fast.next.next;
        }

        if (fast!=null) {                  // used small chatgpt hint tho , to check even or odd
            slow=slow.next;
        }
        while (slow!=null) {
            if (stack.pop() != slow.val) {
                return false;
            }
            slow = slow.next;
        }
        if (!stack.isEmpty()) {
            return false;
        }
        return true;
    }
}
```
