# Tiny Concept

Yesterday you learned Binary Search that finds **one occurrence**.

Today you'll learn something much more important.

> Binary Search isn't just for finding an element.

It's for finding **boundaries**.

Imagine:

```text
1 2 2 2 2 3 4
```

Suppose target = `2`.

Which `2`?

* First?
* Last?
* Any?

Yesterday's Binary Search finds **any** one.

Today's problems teach you how to find the **boundary**.

This idea appears everywhere:

* First occurrence
* Last occurrence
* Lower Bound
* Upper Bound
* Search Insert Position (which you unknowingly already solved)

You're no longer searching for

left boundary

while (low < high)

You stop when:

low == high

At that point, the search window has collapsed to one boundary position.

If nums[mid] >= target, move high = mid
Otherwise, move low = mid + 1

When the loop stops, low is the first position where the value is at least target.

So think of it as two separate searches:

first = left boundary of target
last  = right boundary of target

So what should you do?

Remember answer = mid

Yes.
Where can it be?
Only on the left.

Throw away the right half and continue searching left.

Notice something.

You're not searching because you haven't found the target.

You're searching because you want a better answer.

Example

Suppose first Binary Search lands at index 3.

1 2 2 2 2 3 4
      ^

answer = 3

1 2 2

answer = 1

1

Search ends.

Final answer:

For first occurrence:

Save the current index.
Continue searching left.

For last occurrence:

Save the current index.

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        int low = 0;
        int high = nums.length-1;
        int[] arr = {-1,-1};
        while (low <= high) {
            int mid = (low+high) / 2;
            if (nums[mid] == target) {
                arr[0] = mid;
                high = mid-1;
            }
            else if (target < nums[mid]) {
                high = mid-1;
            }
            else if (target > nums[mid]) {
                low = mid+1;
            }
        }
        low = 0;
        high = nums.length-1;
        while (low <= high) {
            int mid = (low+high) / 2;
            if (nums[mid] == target) {
                arr[1] = mid;
                low = mid+1;
            }
            else if (target < nums[mid]) {
                high = mid-1;
            }
            else if (target > nums[mid]) {
                low = mid+1;
            }
        }
        return arr;
    }
}
```

Time Complexity: O(log n) + O(log n) = O(log n)
Space Complexity: O(1)
