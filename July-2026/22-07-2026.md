```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        if (head == null || head.next == null) {
            return false;
        }
        ListNode slow = head;
        ListNode fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;

}
```

assume 
```
Head
 ↓
A → B → C → D → E
          ↑     ↓
          H ← G ←F
```
then 
x = distance from head to cycle start
y = distance from cycle start to meeting point
z = remaining distance back to cycle start 

so Cycle length = y + z

How far has slow travelled?

Slow moves: x + y

because it walks

head -> cycle start -> meeting point


How far has fast travelled?

Fast moves twice as fast - so 2(x+y) (theoretically)

in practice - x + y + (y + z)

Then: 2(x+y)=x+y+(y+z)

Subtract: x+y=y+z

Therefore: x=z (That equation is the entire trick.)

distance(head, cycle start) = distance(meeting point, cycle start)

so Leave one pointer at Meeting point and Move another back to Head.

Now move both: 1 step

Because x=z, they arrive at the cycle start together.

#### The Algorithm
Phase 1

Detect whether a cycle exists.
Exactly the same as Problem 141.

Phase 2

When slow == fast
don't return.

Instead:
pointer1 = head
pointer2 = meeting point

Move both one step:

pointer1 = pointer1.next
pointer2 = pointer2.next

Eventually:

pointer1 == pointer2

That node is the cycle start.


Should you memorize this?
No.

But you should remember the intuition:

The fast pointer gains exactly one cycle's worth of distance over the slow pointer. That relationship makes the distance from the meeting point to the cycle start equal to the distance from the head to the cycle start.

The algebra (x = z) is something you can revisit if you forget the proof, but the two-phase algorithm and the intuition are what interviewers expect you to know.

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        if (head==null || head.next == null) {
            return null;
        }
        ListNode slow = head;
        ListNode fast = head;
        while (fast!=null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) {
                slow = head;
                break;
            }
        }
        while (fast!=null) {
            if (slow == fast) {
                return slow;
            }
            slow = slow.next;
            fast = fast.next;
        }
        return null;
    }
}
```
