## 📚 Topics Covered

### 1. Classes & Objects

* Defined classes with fields and methods
* Created objects and accessed data

#### Key Learning:

* Objects = real-world representation
* Class = blueprint

---

### 2. Constructors

* Default constructor
* Parameterized constructor
* `this` keyword

#### Key Learning:

* Constructor = object initialization
* `this` helps assign values correctly

---

### 3. Encapsulation

* `private` variables
* Getters & setters

#### Key Learning:

* Data hiding
* Controlled access + validation

---

### 4. Inheritance

* `extends` keyword
* Parent → Child relationship
* Code reuse

#### Key Learning:

* Child inherits properties + behavior
* Reduces duplication

---

### 5. super Keyword

* Calling parent constructor
* Accessing parent data

#### Key Learning:

* Parent constructor must be called
* Execution starts from parent

---

### 6. Method Overriding

* Same method in parent + child
* Child changes behavior

#### Key Learning:

* Behavior depends on object
* `@Override` is optional (but recommended)

---

### 7. Runtime Polymorphism

* Parent reference → Child object

```java
Animal1 a = new Dog1("Rocky");
```

#### Key Learning:

* Compile time → checks reference
* Runtime → executes object method

---

### 8. Method Overloading

* Same method name
* Different parameters

#### Key Learning:

* Compile-time polymorphism
* Cannot overload using only return type

---

### 9. Abstraction

* Abstract classes
* Abstract methods

#### Key Learning:

* Defines structure (what)
* Hides implementation (how)

---

### 10. Interfaces

* Contract-based design
* `implements` keyword
* Multiple inheritance

#### Key Learning:

* Interface = capability (can-do)
* Used for unrelated classes sharing behavior

---

### 11. Static Keyword

* Static variables (shared)
* Static methods (no object required)

#### Key Learning:

* Belongs to class, not object
* Used for utility methods and shared data

---

### 12. Exception Handling

* try-catch blocks
* throw vs throws

#### Key Learning:

* Runtime error handling
* throw = create exception
* throws = delegate handling
